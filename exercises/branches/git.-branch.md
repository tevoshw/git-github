**GIT BRANCH**: See all the branchs in LOCAL repository
![alt text](image.png)
**GIT BRANCH -R:** Show all the branchs in REMOTE repository


**GIT SWITCH 'NAME'**: Change the actual branch
**GIT CHECKOUT 'NAME':** Change the actual branch (better option then switch)

**GIT MERGE 'NAME' -M 'MENSSAGE'**: Commit others branchs

**GIT FETCH ORIGIN**: Pull the adds from origin to the branch
**GIT MERGE:** Get the news from the origin (no show the adds)
